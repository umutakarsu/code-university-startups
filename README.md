# CODE University Berlin - Student Startups Showcase

A beautiful, responsive website showcasing the remarkable entrepreneurial success of CODE University Berlin students and alumni.

## About

CODE University of Applied Sciences, founded in 2017, has established itself as Germany's leading private university for tech talent and technology-driven founders. This website highlights the impressive portfolio of student-founded startups, including:

- **60+ companies** founded by CODE students
- **€250M+** in total funding raised
- **5 Y Combinator** backed startups
- **€100M+** cumulative valuation

## Featured Companies

The website showcases notable startups including:

- **SLAY** - Social gaming platform with 2.5M+ monthly active users ($5M funding, YC-backed)
- **Kombo** - Unified API for HR and recruiting software (€8M funding, YC-backed)
- **Langdock** - GDPR-compliant AI platform ($3M funding, YC-backed)
- **Certus One** - Blockchain infrastructure (acquired by Jump Trading)
- **AULIOS** - AI-powered sound-to-light automation

## Getting Started

### Local Development

Simply open the `index.html` file in your web browser:

```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# Or just double-click the file
```

### Serving with a local server

For a more production-like environment:

**Using Python:**
```bash
# Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```

**Using Node.js:**
```bash
# Install http-server globally
npm install -g http-server

# Run server
http-server

# Then visit http://localhost:8080
```

**Using VS Code:**
- Install the "Live Server" extension
- Right-click on `index.html` and select "Open with Live Server"
